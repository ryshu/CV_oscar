# [Oscar Marie--Taillefer CV](https://ryshu.github.io/cv)

Based on [particle](https://github.com/nrandecker/particle) repository

## CMD

```js
# Dev server
npm run dev

# Build site
npm run dev jekyll-build

# Build js and min
npm run dev js

# Build sass and min
npm run dev sass

# Build fonts and min
npm run dev fonts

# CP image and min
npm run dev imagemin
```

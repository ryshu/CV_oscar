Projets personnels

JSON Path Picker

  • HTML 5
  • HTML 3
  • Javascript
  • VueJS (Vue JSON Picker)

Le sélecteur de chemin JSON est un outil en ligne qui permet de transformer des chaînes JSON en représentations HTML. La principale différence par rapport aux autres visualiseurs JSON est que le sélecteur de chemin JSON permet de trouver un chemin vers la clé en cliquant sur une icône proche du nom de la clé. Cela simplifie le processus de création de JSONPaths en fonction des réponses simulées.
